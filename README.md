# linux-network-toolkit
Network diagnostics, interface management, and connectivity testing.

## Scripts
| Script | Purpose |
|--------|---------|
| `scripts/net_diag.sh` | Full network diagnostic report |
| `scripts/port_scanner.sh` | Check open/closed ports on hosts |
| `scripts/bandwidth_test.sh` | Network bandwidth test with iperf3 |
| `scripts/firewall_audit.sh` | Show and validate firewall rules |
| `scripts/dns_check.sh` | DNS resolution test for all configured servers |
